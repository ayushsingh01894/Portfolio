E-Commerce Product Performance Analysis

A complete data analytics project focused on analyzing e-commerce sales data to uncover trends, profitability insights, and customer behavior patterns. The project includes data cleaning, SQL analysis, exploratory data analysis, and dashboard visualization.

Project Highlights

  • Data cleaning and preprocessing using Excel & Python
  • SQL queries for sales and profit analysis
  • Exploratory Data Analysis (EDA)
  • Data visualization using Matplotlib & Seaborn
  • Interactive Power BI dashboard with KPIs

Technologies Used

  • Python (Pandas, NumPy)
  • Matplotlib
  • Seaborn
  • SQL
  • Excel
  • Power BI
