QR Code–Based Attendance System

A web-based automated attendance tracking system built using Flask. The system generates unique QR codes for students and records attendance securely when scanned. This reduces manual effort and prevents proxy entries.

Project Highlights

  • QR Code generation for students
  • Secure attendance recording system
  • Flask backend with routing & logic
  • Database integration for attendance storage
  • Responsive frontend using HTML, CSS & JavaScript

Technologies Used

  • Python
  • Flask
  • HTML
  • CSS
  • JavaScript
  • SQLite / CSV Database
